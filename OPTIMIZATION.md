# AI 角色聊天演示优化清单 (Concise Checklist)

### 1. 必做项 (PRD 补全)
- [ ] **角色管理**：补全“编辑”功能（目前仅支持增删）。
- [ ] **历史记录**：添加侧边会话列表（支持切换、加载旧消息）。
- [ ] **多模态修复**：恢复图片上传 UI，后端增加更详细的 400 错误日志捕获。

### 2. 演示增强 (极客提分项)
- [ ] **提示词可视化**：前端增加“查看当前提示词”按钮（展示人设拼接逻辑）。
- [ ] **Debug 面板**：实时显示 API 请求/返回的 JSON，展示对数据链路的理解。
- [ ] **打字机优化**：流式输出增加 `cursor` 闪烁动效，让回复更自然。

### 3. 保命项 (演示稳定性)
- [ ] **Mock 模式**：后端增加 `MOCK_MODE` 开关，API 挂掉时返回预设文本，确保演示不中断。
- [ ] **错误柔性处理**：API 报错时前端弹出友好提示，而不是控制台静默报错或界面卡死。
